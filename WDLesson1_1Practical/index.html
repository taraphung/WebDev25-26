Web Technologies

HTML

HTML brings elements to the web page. It uses a system of tags to define elements including headings and paragraphs, which creates a structured layout for web browsers. HTML stands for HyperText Markup Language.

CSS

CSS allows to style and decorate the elements of a web page. This includes changing colors, fonts, spacing, backgrounds, etc. CSS stands for Cascading Style Sheets.

JavaScript

JavaScript allows to control behavior of elements in a web page. It can update and change both HTML and CSS. It can calculate, manipulate and validate data.

JQuery

JQuery is a tool that helps write less code when building websites. It makes things like clicking buttons, hiding text, or loading data easier. People use it to save time and make web pages work better on browsers.
